## Packages
@supabase/supabase-js | Supabase client for authentication
recharts | Data visualization for expense charts
date-fns | Date formatting and manipulation
framer-motion | Smooth animations and page transitions

## Notes
Supabase URL and Key are required in environment variables (VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY).
Authentication is handled via Supabase, but API requests to backend need the access_token in Authorization header.
